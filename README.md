El trabajo está basado un dataset que contiene información de distintas campañas de marketing de una entidad bancaria portuguesa. El problema es de clásificación binaria, donde el objetivo es predecir si un cliente se suscribirá o no a un depósito a plazo fijo. Para eso, se realizó en primero lugar un análisis descriptivo y evaluación de una serie de modelos de clasificación (Random Foreste, Regresión Lineal, XGBOOST y SVM) para predicir la columna 21 que representa si se suscribe al servicio correspondiente.
